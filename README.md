# NLAC Zoning Agenda CSS

This repo contains a CSS rules for the [Stylus](https://addons.mozilla.org/en-US/firefox/addon/styl-us/) browser extension, to make it easier and faster to print agendas from the [North Linden Area Commission email campaign archive](https://us21.campaign-archive.com/home/?u=330bc1a667408c7b6f03e46bb&id=b4deb59789) as handouts for Zoning Committee meetings.
